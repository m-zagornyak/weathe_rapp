☁️ Python Weather Info App

Простий та зручний застосунок для перегляду погоди, створений на Python із використанням Tkinter для графічного інтерфейсу та OpenWeatherMap API для отримання актуальних даних про погоду. Додаток надає точну інформацію про погодні умови для будь-якого міста та має сучасний і зручний інтерфейс.

✨ Можливості
Актуальна інформація про погоду
Відображення температури, атмосферного тиску, вологості, швидкості вітру, а також часу сходу та заходу сонця.
Пошук за містом
Швидко отримуйте інформацію про погоду, ввівши назву будь-якого міста.
Налаштовуваний інтерфейс
Змінюйте кольори, текст та фонові зображення через файл config.py.
Місто за замовчуванням
Під час запуску програма автоматично завантажує погоду для вибраного міста за замовчуванням.
Адаптивний дизайн
Інтерфейс коректно відображається на екранах з різними роздільними здатностями.
Персоналізація фону
Можливість змінювати фонове зображення для створення власного стилю застосунку.
Кросплатформеність
Працює на Windows, macOS та Linux без потреби у додаткових залежностях.
🚀 Встановлення
Крок 1. Клонування репозиторію

Спочатку склонуйте репозиторій на свій комп'ютер:

git clone <URL_репозиторію>
Крок 2. Встановлення необхідних бібліотек

Перейдіть у папку проєкту та встановіть усі залежності:

cd wethe_raoo
pip install -r requirements.txt
Крок 3. Налаштування OpenWeatherMap API

Для отримання даних про погоду необхідно створити API-ключ на сайті OpenWeatherMap.

Після отримання ключа відкрийте файл:

config.py

та додайте свій API-ключ у відповідне поле.

Крок 4. Запуск застосунку

Після налаштування API запустіть програму:

python weather_app.py
Крок 5. Насолоджуйтесь!

Тепер ви можете вводити назву будь-якого міста та миттєво отримувати актуальну інформацію про погоду.

⚙️ Налаштування

Усі параметри застосунку зберігаються у файлі:

config.py

У ньому можна змінювати:

API-ключ OpenWeatherMap;
місто за замовчуванням;
кольори інтерфейсу;
текстові елементи;
фонове зображення;
інші параметри оформлення.

Завдяки цьому ви можете легко персоналізувати застосунок відповідно до власних вподобань.
